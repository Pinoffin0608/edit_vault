# EditVault

A professional terminal-based freelance project management system built with Python, MySQL, and Rich UI.

EditVault helps freelancers, agencies, and developers manage:

- Clients
- Projects
- Payments
- Deadlines
- Earnings
- Project Status Tracking

with a modern interactive terminal interface.

---

# Features

## Authentication System
- Secure login system
- Password hidden using `getpass`
- Retry protection for invalid logins

## Client Management
- Add clients
- View clients
- Search clients
- Update client details
- Delete clients

## Project Management
- Add projects
- Track deadlines
- Update project status
- Search projects
- Delete projects

## Payment Management
- Add payment records
- Track completed/pending payments
- Update payment status
- Delete payment records

## Smart Dashboards

### Earnings Dashboard
- Total earnings
- Monthly earnings
- Yearly earnings
- Completed payment analytics

### Project Dashboard
- Completed projects
- Ongoing projects
- Deadline tracking
- Overdue alerts

## Professional Terminal UI

Built using Rich library:
- Tables
- Panels
- Progress bars
- Colored statuses
- Interactive arrow-key menus

## Auto Alert System
Displays:
- Urgent projects
- Overdue deadlines
- Remaining days
